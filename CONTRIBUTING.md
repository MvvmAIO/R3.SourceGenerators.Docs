# Contributing to R3.SourceGenerators.Docs

This repository is the **VitePress** documentation site for [MvvmAIO.R3.SourceGenerators](https://github.com/MvvmAIO/MvvmAIO.R3.SourceGenerators).

- **Contributor guide:** [docs/contributing.md](docs/contributing.md) (English) / [docs/zh/contributing.md](docs/zh/contributing.md) (简体中文)
- **Live site:** https://mvvmaio.github.io/R3.SourceGenerators.Docs/
- **Prerequisites:** Node.js 22 (`nvm use` with [`.nvmrc`](.nvmrc))

```bash
npm install
npm run docs:dev
```

Generator and analyzer work happens in the **MvvmAIO.R3.SourceGenerators** repository, not here.
