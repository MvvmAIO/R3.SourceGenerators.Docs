# 示例

::: tip 语言
[English](../samples)
:::

可运行示例在独立仓库：

**[MvvmAIO.R3.SourceGenerators.Samples](https://github.com/MvvmAIO/MvvmAIO.R3.SourceGenerators.Samples)**

包含 **WPF**（`UseWPF`）与 **Avalonia** 演示 `FromEvents`、`FromRoutedEvents` 与 `[R3Command]`。

克隆后打开 **`MvvmAIO.R3.SourceGenerators.Samples.slnx`**，按示例仓库说明使用 NuGet 或本地分析器包。

## 本地分析器与 NuGet

| 方式 | 场景 |
|------|------|
| **NuGet**（默认） | 对照已发布包运行演示 |
| **本地 pack / 源** | 发版前验证生成器改动 |

步骤见 [示例仓库 README](https://github.com/MvvmAIO/MvvmAIO.R3.SourceGenerators.Samples/blob/master/README.md)（含 `dotnet nuget locals`、restore 与本地包），此处不重复。
